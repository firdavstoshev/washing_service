## Запуск проекта

### 1. Клонируйте репозиторий

```sh
git clone https://github.com/firdavstoshev/washing_service.git
```

```shell
cd washing_service
```

### 2. Запустите контейнеры

```shell
docker compose up -d
```

### 3. Перейдите в браузере по адресу

```shell
http://localhost:8080/swagger/index.html
```
